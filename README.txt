Ran on "Render"
Built for the WAD2 coursework

Setup:

- Sign in to render
- Create new project and choose "Web Services"
- Connect to Github
- Pick the repository or choose "Public Git Repository" 
  and use "https://github.com/Ax3Liar/WAD2.git"
- Choose "node" on language if not done already
- On "Build Command" put: npm install
- On "Start Command" put: node seed/seed.js && npm start
- generate/create an environment variable
- choose "Instance type" (free should work)
- "Deploy Web Service" at the bottom of the page
